# Links to Add on Google Sites (nlp.hanyang.ac.kr)

Base URL: **https://galsang.github.io/hyu-nlp-publications/**

## How to Add Links on Google Sites

1. Go to https://sites.google.com and open your site
2. Navigate to the **Publications** page
3. Click the pencil icon to enter **Edit mode**
4. For each paper, add a button or text link:
   - **Option A (Button):** Insert → Button → paste the URL below → label it "Detailed Explanation"
   - **Option B (Text link):** Select the paper title text → click the link icon (🔗) → paste the URL below
5. Click **Publish** when done
