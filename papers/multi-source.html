Analysis of Multi-Source Language Training in Cross-Lingual Transfer

ACL 2024
Seong Hoon Lim, Taejun Yun, Jinhyeon Kim, Jihun Choi, Taeuk Kim

One-Line Summary

An analysis of training strategies using multi-source language data for cross-lingual transfer

Key Points

  • Systematically analyzes the effects and optimal strategies when training models by combining language data from multiple sources
  • Empirically investigates how interactions between data sources affect model performance
