A Cross-Sentence Latent Variable Model for Semi-Supervised Text Sequence Matching

ACL 2019
Jihun Choi, Taeuk Kim, Sang-goo Lee

One-Line Summary

Semi-supervised text sequence matching using a cross-sentence latent variable model

Background & Motivation

Text sequence matching -- determining whether two text sequences are semantically related, entailing, or paraphrases of each other -- is a fundamental NLP task. Supervised models require large labeled datasets, which are expensive to obtain. Semi-supervised approaches can help, but most treat each sentence pair independently, ignoring useful structural dependencies across the dataset.

Limitation: Existing semi-supervised matching models encode each sentence pair in isolation and fail to capture cross-instance latent patterns that could provide additional supervisory signal from unlabeled data.

Innovation: This work introduces cross-sentence latent variables that capture shared structure across sentence pairs, enabling the model to propagate information from labeled examples to unlabeled ones through a shared latent space.

Method

The proposed model augments a standard text matching architecture with a latent variable that spans across sentence pairs. Using variational inference, the model learns a shared latent space that captures dataset-level distributional patterns, allowing unlabeled data to contribute meaningful learning signal.

  • Cross-Sentence Latent Variables: A latent variable model that captures dependencies not just within a sentence pair, but across multiple pairs in the dataset
  • Variational Inference: An inference network approximates the posterior over latent variables, enabling end-to-end training with both labeled and unlabeled data
  • Semi-Supervised Objective: Combines a supervised matching loss on labeled data with an unsupervised reconstruction/regularization loss on unlabeled data through the shared latent space

Results

  • Natural Language Inference: Consistent improvements on NLI benchmarks in semi-supervised settings, demonstrating the value of cross-sentence latent structure
  • Paraphrase Detection: Enhanced matching accuracy when only a fraction of training data is labeled, outperforming standard semi-supervised baselines
  • Label Efficiency: The latent variable approach shows the largest gains in low-label regimes, confirming its effectiveness at extracting information from unlabeled data

Why It Matters

This work demonstrated that modeling cross-instance dependencies through latent variables is a powerful strategy for semi-supervised text matching. By learning a shared latent space that captures dataset-level patterns, the model can transfer knowledge from labeled to unlabeled examples more effectively than approaches that treat each pair in isolation. This idea of leveraging structural dependencies across examples remains relevant in modern few-shot and low-resource NLP settings.
