RAISE: Enhancing Scientific Reasoning in LLMs via Step-by-Step Retrieval

The 5th Workshop on Mathematical Reasoning and AI (MATH-AI) at NeurIPS 2025
Minhae Oh, Jeonghye Kim, Nakyung Lee, Donggeon Seo, Taeuk Kim, Jungwoo Lee

One-Line Summary

A retrieval-augmented framework that enhances multi-step scientific reasoning in LLMs by decomposing problems and retrieving logically relevant documents at each reasoning step, rather than relying on single-query surface-level retrieval.

Background & Motivation

Scientific reasoning tasks require LLMs to handle long-chain reasoning processes alongside domain-specific terminology and up-to-date knowledge. Conventional RAG approaches retrieve documents using a single query, which often returns vague or superficially related content that fails to support the multi-step logic needed for graduate-level science questions.

Key Problem: Existing retrieval methods (e.g., standard RAG, HyDE) match documents based on surface similarity rather than logical relevance, leading to retrieved passages that share domain keywords but lack the essential scientific mechanisms needed for reasoning.

Proposed Method: RAISE

RAISE operates through three sequential phases to retrieve logically relevant documents from unstructured corpora:

1
Problem Decomposition
Complex scientific queries are broken down into structured subquestions paired with initial search queries, creating a foundation for step-aware retrieval
2
Logical Query Generation
Initial queries are reformulated into logic-enriched versions by combining both the subquestion and original search query, capturing reasoning intent beyond superficial domain similarity
3
Logical Retrieval
Documents matching each reformulated query are retrieved from open-domain sources (e.g., Wikipedia) using a similarity threshold, supporting sequential subanswer generation

Experimental Results

RAISE is evaluated on multiple graduate-level scientific reasoning benchmarks, consistently outperforming baseline approaches across different LLM scales (LLaMA-8B, GPT-4o mini, Mistral-24B).

Why It Matters

Scientific reasoning is one of the most challenging frontiers for LLMs, requiring both domain knowledge and multi-step logical deduction. RAISE demonstrates that step-wise decomposition combined with logic-enriched retrieval can substantially improve performance on graduate-level science questions, offering a practical framework for building more capable scientific reasoning systems without additional model training.
