Enhancing Conversational Context Inference via Reasoning Feedback-Based Learning

The 37th Annual Conference on Human and Cognitive Language Technology (HCLT 2025)
Yuri Son, Taeuk Kim

One-Line Summary

A novel reasoning feedback-based learning (RFL) approach that leverages external model feedback to progressively improve a language model's ability to infer conversational context.

Background

Understanding conversational context — inferring implicit information, speaker intentions, and situational dynamics from dialogue — is a fundamental challenge in dialogue systems. While large language models have improved dialogue understanding, they still struggle with complex inference cases that require multi-step reasoning over conversation history. Standard fine-tuning approaches often plateau on difficult examples where surface-level pattern matching fails, leaving a significant portion of challenging conversational inferences unresolved.

Method

  • Reasoning Feedback-Based Learning (RFL): Proposes a learning framework where an external, more capable model generates detailed reasoning feedback on the target model's incorrect predictions, providing explanations of why certain inferences are wrong and what the correct reasoning path should be.
  • Progressive data refinement: Uses the reasoning feedback to iteratively improve training data quality, focusing on cases where the model previously failed, thereby concentrating learning effort on the most challenging examples.
  • Feedback-guided fine-tuning: Incorporates the structured reasoning feedback into the training process, enabling the model to learn not just correct answers but also the reasoning patterns behind them.

Results

  • The RFL approach achieves 95.04% accuracy on the conversational context inference task, surpassing the baseline by 7.93 percentage points.
  • RFL outperforms standard multiple-choice question (MCQ) fine-tuning by 1.32 percentage points, demonstrating the added value of reasoning feedback over conventional training.
  • Detailed feedback from more powerful external models plays a crucial role in overcoming the most challenging inference cases that standard methods fail to resolve.

Significance

This work introduces a practical methodology for improving dialogue understanding through structured reasoning feedback, showing that external model guidance can break through performance plateaus on hard conversational inference examples. The progressive refinement approach offers a general framework applicable beyond dialogue to any task where reasoning quality matters.
