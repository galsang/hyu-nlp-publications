Cell-aware Stacked LSTMs for Modeling Sentences

ACML 2019
Jihun Choi, Taeuk Kim, Sang-goo Lee

One-Line Summary

Improving sentence modeling with cell-aware stacked LSTMs

Background

Stacked (multi-layer) LSTMs are a standard architecture for sequence modeling, with each layer processing the hidden states of the layer below. However, a critical design limitation exists: while hidden states are passed between layers, the cell states — the internal memory that gives LSTMs their ability to capture long-range dependencies — are not communicated across layers. Each layer maintains an independent cell state, meaning that the rich memory accumulated in lower layers is invisible to higher layers. This information bottleneck limits the representational capacity of deep LSTM networks.

Method

  • Cell-aware inter-layer connections: The paper introduces a modified stacked LSTM architecture where the cell state of the previous layer is explicitly fed into the gating mechanisms of the current layer. This allows upper layers to read and leverage the long-term memory stored in lower layers.
  • Minimal architectural overhead: The modification is lightweight — it augments the existing gate computations with an additional cell-state input rather than introducing entirely new modules, keeping the parameter increase modest.
  • Drop-in replacement: Cell-aware stacked LSTMs can serve as a direct replacement for standard stacked LSTMs in any existing pipeline, requiring no changes to the overall training procedure or loss functions.

Results

  • Consistent improvements: The cell-aware architecture yields consistent gains over standard stacked LSTMs across multiple sentence-level tasks including natural language inference, sentiment analysis, and semantic relatedness.
  • Effective memory utilization: Analysis shows that upper layers in the cell-aware variant produce richer representations by incorporating memory signals from lower layers, confirming that inter-layer cell communication alleviates the information bottleneck.
  • Scalability with depth: The benefits become more pronounced as the number of stacked layers increases, indicating that cell-aware connections are particularly valuable for deeper architectures.

Why It Matters

This work addresses a fundamental but often overlooked limitation in how multi-layer LSTMs are constructed. By enabling cross-layer cell state communication with minimal overhead, it provides a simple yet effective way to build deeper and more expressive recurrent networks. The approach is broadly applicable to any task that uses stacked LSTMs, offering a practical architectural improvement that enhances sentence representations without requiring complex training modifications.
