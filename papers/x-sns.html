X-SNS: Cross-Lingual Transfer Prediction through Sub-Network Similarity

EMNLP 2023 Findings
Taejun Yun, Jinhyeon Kim, Deokyeong Kang, Seong Hoon Lim, Jihoon Kim, Taeuk Kim

One-Line Summary

A method for predicting cross-lingual transfer learning performance

Background

Cross-lingual transfer (XLT) enables multilingual language models to perform well on tasks in unseen languages without target-language labeled data. While English is typically used as the default source language, recent evidence shows that selecting the most appropriate source language for a given target can significantly amplify transfer effectiveness. However, existing methods for predicting transfer compatibility rely on external linguistic resources or typological databases, limiting their practical applicability. A model-oriented approach that directly examines how a multilingual model internally represents different languages could offer a more principled and accessible solution.

Method

X-SNS proposes using sub-network similarity between language pairs as a proxy for predicting cross-lingual transfer compatibility, grounded in the internal structure of the model itself.

1
Sub-Network Extraction
For each language, identify task-relevant sub-networks within the multilingual model by analyzing which parameters are most activated when processing that language's text
2
Similarity Computation
Measure the structural overlap between sub-networks of source and target language pairs -- higher overlap indicates the model processes both languages through similar internal pathways
3
Transfer Prediction
Rank candidate source languages by sub-network similarity to the target language, predicting that higher similarity leads to better transfer performance

Key advantage: The method requires only moderate amounts of raw text from candidate languages -- no labeled data, external linguistic databases, or typological annotations needed.

Results

  • Ranking quality: Achieves an average improvement of 4.6% in NDCG@3 for zero-shot XLT source language ranking over baseline methods
  • Task generalization: Demonstrates consistent superiority across diverse NLP tasks, confirming that sub-network similarity is a robust predictor of transfer compatibility
  • Model-internal insight: Extensive analyses confirm that languages sharing more sub-network structure in the multilingual model indeed transfer knowledge more effectively

Why It Matters

X-SNS provides a practical, model-grounded mechanism for source language selection in cross-lingual transfer. Unlike methods that depend on external typological knowledge, X-SNS looks directly at how the model internally represents languages, making it applicable to any language with raw text data. This is particularly valuable for deploying multilingual systems efficiently, as choosing the right source language can make the difference between successful and failed cross-lingual transfer -- especially for low-resource languages where every design decision matters.
