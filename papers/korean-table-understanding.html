Learning Strategies to Improve Table Understanding and Explanation in Korean

The 36th Annual Conference on Human and Cognitive Language Technology (HCLT 2024) Best Paper Award
Changhyeon Kim, Seunghee Kim, Taeuk Kim

One-Line Summary

A systematic exploration of preprocessing formats and learning strategies to enhance language models' ability to understand and generate natural descriptions of table cells in Korean.

Background

Tables are a ubiquitous format for structured data, yet generating natural language descriptions of specific table cells remains challenging for language models — particularly in Korean. Given a table and a target cell, the task is to produce a fluent Korean description explaining what the cell value represents in context. This capability is essential for table-based question answering, accessibility tools, and automated report generation. However, the choice of how to serialize tables into text (the preprocessing format) and what auxiliary training tasks to use can dramatically affect performance.

Method

  • Table format comparison: Evaluates four serialization formats — Markdown, HTML, Pandas DataFrame, and JSON — to determine which representation most effectively conveys table structure to language models.
  • Auxiliary subtasks with synthetic data: Designs four targeted subtasks — cell position recognition, row information recognition, column information recognition, and table structure rearrangement — each supported by custom synthetic datasets to build table comprehension skills incrementally.
  • Token-efficient training: Identifies preprocessing formats that minimize token count while preserving structural information, reducing computational costs during both training and inference.

Results

  • Pandas DataFrame format achieves the best overall performance, improving baseline results by 19.6% while using fewer tokens than HTML or Markdown representations.
  • The auxiliary subtasks with synthetic data provide consistent gains, with cell position recognition and column information recognition contributing most to the final table description quality.
  • Token-efficient formats not only reduce computational cost but also improve model performance, suggesting that concise structural representations help models focus on relevant information.

Significance

This work provides practical, actionable guidelines for building Korean table understanding systems. By demonstrating that the right combination of preprocessing format and targeted auxiliary training can yield substantial improvements, it establishes a strong foundation for table-to-text generation in Korean. The paper received the Best Paper Award at HCLT 2024, recognizing its contributions to an underexplored but practically important area of Korean NLP.
