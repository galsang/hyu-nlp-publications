Enhanced Prompting for Multi-Label Classification

Korea Computer Congress 2024 (KCC 2024)
Jungyeon Lee, Youngwoo Shin, Yejin Yoon, Taeuk Kim

One-Line Summary

An enhanced prompting framework that improves large language models' ability to handle multi-label classification by refining prompt design strategies for in-context learning.

Background

Multi-label classification — where a single input can be assigned multiple labels simultaneously — is a common requirement in real-world NLP tasks such as topic tagging, emotion detection, and intent recognition. While large language models (LLMs) have shown strong performance on single-label tasks via in-context learning (ICL), their effectiveness on multi-label scenarios remains limited. Standard prompting strategies often struggle to guide models toward predicting the correct set of labels, especially when the label space is large and label combinations are diverse.

Method

  • Prompt restructuring: Redesigns the prompt format to better convey the multi-label nature of the task, explicitly guiding the model to consider multiple labels per input rather than defaulting to single-label prediction.
  • Demonstration selection: Proposes strategies for selecting in-context examples that effectively illustrate multi-label assignments, ensuring diversity in label combinations within the demonstration set.
  • Label verbalization: Investigates how different ways of presenting label names and descriptions in the prompt affect the model's ability to predict multiple relevant labels.

Results

  • The proposed enhanced prompting strategies consistently outperform naïve prompting baselines across multiple multi-label classification benchmarks.
  • Carefully selected demonstrations with diverse label combinations significantly improve prediction quality compared to random demonstration selection.
  • The approach is effective across different LLM architectures without requiring any fine-tuning or parameter updates.

Significance

This work demonstrates that thoughtful prompt engineering can substantially close the performance gap between LLMs and supervised models on multi-label tasks. By providing a systematic framework for enhanced prompting in the multi-label setting, it extends the practical applicability of in-context learning to a broader class of classification problems encountered in production NLP systems.
