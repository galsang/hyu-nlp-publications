A Framework for Narrative Structure Analysis Using Automated Semantic Role Labeling in Korean Text

Eun Rang Kwon, Junmo Song, Donggeon Seo, Kangmin Lee, Taeuk Kim, Jeong-Han Kang

Summary

This paper presents a framework for analyzing narrative structures in Korean text using automated semantic role labeling, bridging computational linguistics and sociological text analysis.
