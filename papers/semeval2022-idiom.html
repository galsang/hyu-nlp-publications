HYU at SemEval-2022 Task 2: Effective Idiomaticity Detection with Consideration at Different Levels of Contextualization

International Workshop on Semantic Evaluation (SemEval 2022) at NAACL 2022
Youngju Joung, Taeuk Kim

One-Line Summary

A unified framework for idiomaticity detection that leverages four features computed at different levels of contextualization -- both inter-sentence and inner-sentence context -- to determine whether a multi-word expression is used idiomatically or literally.

Background & Motivation

Multi-word expressions (MWEs) like "wet blanket" can be interpreted either compositionally ("a piece of cloth soaked in liquid") or idiomatically ("a person who spoils the mood"). Detecting whether an MWE is used idiomatically is a challenging NLP problem because the same expression can have different meanings depending on context.

Key Challenge: Prior work showed that simply concatenating surrounding context sentences is generally unhelpful for idiomaticity detection, and can even degrade performance by tripling input sequence length. A more sophisticated approach to exploiting context is needed.

Proposed Method: Multi-Level Contextualization Framework

The framework computes four features from a Transformer encoder (XLM-R), each capturing different aspects of contextualization, then concatenates them for a linear classifier:

1
Surrounding Context Features
Two features based on inter-sentence context: the target sentence is paired separately with its previous and next sentences, using segment embeddings to distinguish MWE tokens. This avoids overwhelming the encoder while emphasizing the target sentence.
2
Inner-Sentence Context Features
Two features inspired by metaphor detection theories (MIP and SPV): a context-exclusive representation (target sentence with MWE masked) and an MWE-exclusive representation (MWE alone), capturing the difference between contextualized and static semantics.

Experimental Results

Evaluated on SemEval-2022 Task 2 (idiomaticity detection for two-noun compounds) across three languages: English, Portuguese, and Galician, in both zero-shot and one-shot settings.

  • Strong Zero-shot Performance: Outperforms XLM-R baseline by over 16% on Galician (unseen language), demonstrating strong cross-lingual generalization
  • State-of-the-art One-shot: Achieves 89.96 overall macro F1 in the one-shot setting after post-evaluation refinement
  • Context Helps: Separating surrounding context into two chunks (previous + target, target + next) is consistently advantageous over no-context and naive concatenation approaches
  • Stable Performance: The proposed approach shows much smaller score deviation and more stable results than baselines

Why It Matters

Understanding idiomatic language is essential for NLP systems to properly process figurative expressions. This work demonstrates that carefully designed multi-level contextualization -- drawing from both surrounding context and metaphor detection theories -- can significantly improve idiomaticity detection, especially for languages not seen during training.
