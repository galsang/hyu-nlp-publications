Dynamic Compositionality in Recursive Neural Networks with Structure-aware Tag Representations

AAAI 2019
Taeuk Kim, Jihun Choi, Daniel Edmiston, Sanghwan Bae, Sang-goo Lee

One-Line Summary

Dynamic compositionality in recursive neural networks using structure-aware tag representations

Key Points

  • Introduces structure-aware tag representations into recursive neural networks to implement dynamic composition operations guided by syntactic structure
  • Improves sentence representation quality by adaptively adjusting composition functions according to the structural characteristics of the input
