Dynamic Compositionality in Recursive Neural Networks with Structure-aware Tag Representations

AAAI 2019
Taeuk Kim, Jihun Choi, Daniel Edmiston, Sanghwan Bae, Sang-goo Lee

One-Line Summary

Dynamic compositionality in recursive neural networks using structure-aware tag representations

Background

Recursive neural networks (RecNNs) build sentence representations by composing word vectors bottom-up along a syntactic parse tree. A key limitation of standard RecNNs is that they apply the same composition function at every node — whether combining a determiner with a noun or merging two clauses. Linguistically, however, different syntactic constructions call for different composition operations. This mismatch limits the expressiveness of traditional recursive models and motivates the need for dynamic composition that adapts to the syntactic context at each tree node.

Method

  • Structure-aware tag representations: Each node in the parse tree is augmented with a learned representation of its syntactic tag (e.g., NP, VP, S). These tag embeddings encode the structural role of each constituent and serve as conditioning signals for the composition function.
  • Dynamic composition functions: Instead of a single fixed weight matrix, the composition function at each node is dynamically generated based on the tag representations of the children and parent. This allows the model to apply different transformations for different syntactic configurations (e.g., adjective-noun vs. subject-verb).
  • Seamless integration: The approach is compatible with existing RecNN variants (TreeLSTM, TreeGRU) and can be applied on top of any parse tree, requiring only the addition of tag embeddings and a lightweight parameter generation network.

Results

  • Improved sentence representations: Dynamic compositionality yields consistent improvements over standard RecNNs on sentiment analysis (SST) and natural language inference (SNLI), demonstrating that structure-aware composition produces higher-quality sentence embeddings.
  • Qualitative analysis: Visualization of the learned composition functions shows that the model learns distinct operations for different syntactic patterns, confirming that the dynamic mechanism captures meaningful linguistic distinctions.
  • Complementary to tree quality: Gains hold across both gold parse trees and automatically generated parses, indicating that dynamic compositionality is robust to parse errors.

Why It Matters

This work bridges a gap between linguistic theory and neural network design by making recursive composition syntactically aware. Rather than treating all tree nodes identically, the model adapts its computation to the local syntactic context, producing richer and more linguistically grounded sentence representations. The approach demonstrates that injecting even simple structural signals into neural architectures can yield meaningful improvements, offering a principled direction for building syntax-sensitive NLP models.
