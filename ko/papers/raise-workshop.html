RAISE: Enhancing Scientific Reasoning in LLMs via Step-by-Step Retrieval

The 5th Workshop on Mathematical Reasoning and AI (MATH-AI) at NeurIPS 2025
Minhae Oh, Jeonghye Kim, Nakyung Lee, Donggeon Seo, Taeuk Kim, Jungwoo Lee

한줄 요약

단일 쿼리의 표면적 검색 대신, 문제를 분해하고 각 추론 단계마다 논리적으로 관련된 문서를 검색하여 LLM의 다단계 과학적 추론을 향상시키는 검색 증강 프레임워크입니다.

배경 및 동기

과학적 추론 과제는 LLM이 전문 용어 및 최신 지식과 함께 긴 추론 사슬을 처리해야 합니다. 기존 RAG 접근법은 단일 쿼리로 문서를 검색하여, 대학원 수준의 과학 문제에 필요한 다단계 논리를 지원하지 못하는 모호하거나 피상적인 내용을 반환하는 경우가 많습니다.

핵심 문제: 기존 검색 방법(표준 RAG, HyDE 등)은 논리적 관련성이 아닌 표면적 유사성을 기반으로 문서를 매칭하여, 도메인 키워드는 공유하지만 추론에 필요한 핵심 과학적 메커니즘이 부족한 문서를 검색합니다.

제안 방법: RAISE

RAISE는 비정형 코퍼스에서 논리적으로 관련된 문서를 검색하기 위해 세 단계의 순차적 과정을 수행합니다:

1
문제 분해 (Problem Decomposition)
복잡한 과학 질문을 구조화된 하위 질문과 초기 검색 쿼리로 분해하여 단계별 검색의 기반을 구축
2
논리적 쿼리 생성 (Logical Query Generation)
하위 질문과 원래 검색 쿼리를 결합하여 초기 쿼리를 논리가 강화된 버전으로 재구성, 피상적 도메인 유사성을 넘어 추론 의도를 포착
3
논리적 검색 (Logical Retrieval)
재구성된 각 쿼리에 대해 유사도 임계값을 사용하여 오픈 도메인 소스(예: Wikipedia)에서 문서를 검색하고, 순차적으로 하위 답변 생성을 지원

실험 결과

RAISE는 다양한 대학원 수준 과학적 추론 벤치마크에서 평가되었으며, 다양한 LLM 규모(LLaMA-8B, GPT-4o mini, Mistral-24B)에서 일관되게 기존 방법을 능가합니다.

  • 일반화: 다양한 LLM 규모와 모델 패밀리에서 성능 향상이 유지됨

의의

과학적 추론은 도메인 지식과 다단계 논리적 추론을 모두 필요로 하는 LLM의 가장 도전적인 영역 중 하나입니다. RAISE는 단계별 분해와 논리 강화 검색의 결합이 대학원 수준의 과학 문제에서 성능을 크게 향상시킬 수 있음을 보여주며, 추가적인 모델 훈련 없이 더 강력한 과학적 추론 시스템을 구축하기 위한 실용적인 프레임워크를 제공합니다.
