RAISE: Enhancing Scientific Reasoning in LLMs via Step-by-Step Retrieval

MATH-AI @ NeurIPS 2025
Minhae Oh, Jeonghye Kim, Nakyung Lee, Donggeon Seo, Taeuk Kim, Jungwoo Lee

한줄 요약

단계별 검색을 활용하여 LLM의 과학적 추론 능력을 향상시키는 방법

배경 및 동기

과학적 추론은 긴 논리 체인의 추론뿐만 아니라 도메인 특화 용어에 대한 지식과 지속적으로 업데이트되는 연구 결과에 대한 적응력을 필요로 합니다. 기존 검색 증강 생성(RAG) 접근법은 표면적 유사성에 기반하여 문서를 검색하는데, 이는 논리적으로 관련된 지식이 쿼리와 다른 용어를 사용하는 과학 문제에서 자주 실패합니다. RAISE는 복잡한 과학 문제를 분해하고 단순한 텍스트 유사성이 아닌 논리적 관련성에 기반하여 문서를 검색하는 단계별 검색 프레임워크를 도입하여 이 문제를 해결합니다.

방법론: 3단계 프레임워크

  • 1단계 -- 문제 분해: 과학 문제를 관리 가능한 하위 질문으로 분해하고, 각 하위 질문에 필요한 특정 지식을 포착하는 목표 지향적 검색 쿼리를 생성합니다.
  • 2단계 -- 논리적 쿼리 생성: 각 하위 질문에 대해 문제의 추론 구조를 인코딩하는 논리 강화 쿼리를 생성하여, 키워드 기반 검색을 넘어섭니다.
  • 3단계 -- 논리적 검색: 논리적 관련성에 기반하여 비구조화된 코퍼스에서 각 하위 질문에 관련된 문서를 검색하여, 전체 추론 체인을 지원하는 단계별 증거를 제공합니다.

실험 결과

RAISE는 과학적 추론 벤치마크에서 기준선 방법들을 일관되게 능가합니다. 핵심적인 차별점은 도메인 지식에서 유사할 뿐만 아니라 필요한 추론 단계에 논리적으로 더 관련된 문서를 검색하는 능력으로, 더 정확하고 근거가 탄탄한 답변을 도출합니다.

의의

복잡한 과학 문제는 여러 전문 지식을 활용하는 추론 체인을 요구합니다. RAISE는 문제를 분해하고 단계별로 증거를 검색함으로써, 과학 도메인에서 LLM을 증강하는 원칙적인 방법을 제시합니다 -- 교육, 연구 보조, 자동화된 과학적 발견에 폭넓은 응용 가능성을 가집니다.
