원천 언어 다각화를 통한 교차 언어 전이 성능 향상

Korea Computer Congress 2023 (KCC 2023)
Seong Hoon Lim, Taeuk Kim

한줄 요약

영어에만 의존하지 않고 다양한 원천 언어를 활용하여 제로샷 교차 언어 전이 성능을 향상시키는 방법을 제안하며, 보다 견고한 다국어 표현 학습을 달성합니다.

배경

제로샷 교차 언어 전이는 일반적으로 mBERT나 XLM-R 같은 다국어 사전학습 모델을 단일 원천 언어(거의 항상 영어)의 라벨 데이터로 미세조정한 후, 목표 언어에 대해 별도의 지도학습 없이 평가하는 방식입니다. 이러한 영어 중심 패러다임은 효과적이지만, 영어 특유의 언어 구조에 대한 편향을 도입할 수 있으며, 유형론적으로 먼 언어에 대해서는 최적의 일반화가 어려울 수 있습니다. 본 연구는 영어만이 최선의 원천 언어라는 가정에 의문을 제기하고, 원천 언어 풀을 다각화함으로써 전이 성능을 개선할 수 있는지 탐구합니다.

방법

  • 다중 원천 학습: 영어만을 원천 언어로 사용하는 대신, 다양한 언어의 라벨 데이터를 동시에 활용하여 모델을 미세조정합니다.
  • 언어 선택 전략: 유형론적 다양성(다른 어족, 어순, 형태론 체계)을 최대화하는 원천 언어를 선택하여 모델이 더 넓은 범위의 언어적 현상에 노출되도록 합니다.
  • 통합 미세조정: 결합된 다중 원천 데이터에 대해 단일 다국어 모델을 학습하여, 언어 쌍별 별도 모델의 필요성을 제거합니다.

결과

다각화된 원천 언어로 학습할 경우, 영어만 사용한 기준선 대비 목표 언어에 대한 제로샷 전이 성능이 일관되게 향상되었으며, 특히 영어와 유형론적으로 먼 언어에서 개선이 두드러졌습니다. 깊은 구문적 또는 의미적 이해가 필요한 과제에서 성능 향상이 가장 크게 나타나, 학습 중 다양한 언어 구조에 대한 노출이 보다 언어 독립적인 표현 구축에 도움이 됨을 보여줍니다.

의의

본 연구는 일반적인 영어 단일 원천 학습이 교차 언어 전이에 있어 차선이라는 증거를 제시합니다. 원천 언어 다각화의 이점을 입증함으로써, 비영어 자원을 버리지 않고 기존의 다국어 라벨 데이터를 활용하는 실용적이고 간단한 다국어 NLP 시스템 개선 경로를 제안합니다.
