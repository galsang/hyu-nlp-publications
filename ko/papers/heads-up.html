Heads-up! Unsupervised Constituency Parsing via Self-Attention Heads

AACL-IJCNLP 2020
Bowen Li, Taeuk Kim, Reinald Kim Amplayo, Frank Keller

한줄 요약

셀프 어텐션 헤드를 활용한 비지도 구문 분석 기법

배경

구문 분석(constituency parsing)은 문장을 중첩된 하위 구(phrase)로 분해하는 작업으로, 전통적으로 비용이 큰 트리뱅크 주석이 필요합니다. 비지도 구문 분석은 레이블 데이터 없이 이러한 계층적 구조를 복원하는 것을 목표로 합니다. 한편, BERT와 같은 사전학습 트랜스포머는 내부 표현에 놀라울 정도로 풍부한 구문 정보를 인코딩하는 것으로 알려져 있습니다. 이 모델들의 셀프 어텐션 패턴을 직접 활용하여 명시적 구문 지도 학습 없이도 구문 트리를 유도할 수 있는지가 핵심 질문입니다.

방법

  • 어텐션 기반 스팬 점수: 사전학습 트랜스포머의 선택된 셀프 어텐션 헤드에서 어텐션 가중치를 집계하여 가능한 모든 텍스트 스팬에 대한 구문 친화도 점수를 계산합니다. 높은 점수는 스팬 내 토큰들이 외부 토큰보다 서로에게 더 많이 주의를 기울이고 있어 일관된 구성소임을 나타냅니다.
  • 헤드 선택: 모든 어텐션 헤드가 구문을 동일하게 인코딩하지 않습니다. 본 논문은 위치적 또는 의미적 패턴이 아닌 구문적으로 가장 유용한 헤드를 식별하는 헤드 선택 전략을 도입합니다.
  • 탐욕적 트리 구축: 스팬 점수를 기반으로 탐욕적 하향식 알고리즘이 문장을 재귀적으로 구성소로 분할하여, 레이블 학습 데이터 없이 완전한 이진 구문 트리를 생성합니다.

결과

  • 경쟁력 있는 비지도 구문 분석: Penn Treebank(PTB) 벤치마크에서 더 복잡한 생성 모델이나 특수 학습 목표에 의존하는 기존 비지도 파서와 비교하여 경쟁력 있는 F1 점수를 달성합니다.
  • 헤드 수준 분석: BERT의 중간 레이어에 있는 소수의 어텐션 헤드가 일관되게 구문 구성소 정보를 인코딩하고, 다른 헤드들은 다른 언어적 현상에 집중하고 있음을 상세히 분석합니다.
  • 견고성: 다양한 사전학습 모델에 걸쳐 일반화되며, 태스크별 파인튜닝 없이도 안정적인 성능을 보입니다.

의의

본 연구는 트랜스포머 셀프 어텐션 헤드가 사전학습 과정에서 계층적 구문 구조를 암묵적으로 학습한다는 구체적 증거를 제공합니다. 어텐션 패턴만으로 구문 트리를 추출할 수 있음을 보여줌으로써, 주석 없이도 가능한 경량 구문 분석의 길을 열었습니다. 또한 사전학습 언어 모델의 각 구성 요소에 어떤 언어 지식이 인코딩되어 있는지에 대한 이해를 심화시켜, 향후 모델 설계 및 해석 가능성 연구에 기여합니다.
