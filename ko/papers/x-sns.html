X-SNS: Cross-Lingual Transfer Prediction through Sub-Network Similarity

EMNLP 2023 Findings
Taejun Yun, Jinhyeon Kim, Deokyeong Kang, Seong Hoon Lim, Jihoon Kim, Taeuk Kim

한줄 요약

서브네트워크 유사도를 통해 교차 언어 전이 성능을 예측하는 방법

배경

교차 언어 전이(XLT)는 다국어 언어 모델이 미세조정 시 보지 못한 언어의 태스크에서도 좋은 성능을 발휘할 수 있게 합니다. 영어가 기본 소스 언어로 사용되지만, 최근 연구에 따르면 가장 적합한 소스 언어를 선택하면 전이 효과가 크게 증폭될 수 있습니다. 그러나 기존의 전이 호환성 예측 방법들은 외부 언어학적 자원이나 유형론적 데이터베이스에 의존하여 실용성이 제한됩니다. 다국어 모델이 내부적으로 언어를 어떻게 표현하는지 직접 분석하는 모델 중심 접근법이 더 원칙적이고 접근 가능한 해결책이 될 수 있습니다.

방법

X-SNS는 모델 내부 구조에 기반하여, 언어 쌍 간의 서브네트워크 유사도를 교차 언어 전이 호환성 예측의 대리 지표로 활용합니다.

1
서브네트워크 추출
각 언어에 대해, 해당 언어의 텍스트를 처리할 때 가장 활성화되는 파라미터를 분석하여 다국어 모델 내 태스크 관련 서브네트워크를 식별
2
유사도 계산
소스 언어와 타겟 언어 쌍의 서브네트워크 간 구조적 중첩을 측정 -- 중첩이 클수록 모델이 두 언어를 유사한 내부 경로로 처리함을 의미
3
전이 성능 예측
타겟 언어와의 서브네트워크 유사도를 기준으로 후보 소스 언어를 순위 매김, 유사도가 높을수록 더 나은 전이 성능을 예측

핵심 장점: 후보 언어의 적당한 양의 원시 텍스트만 필요하며, 레이블된 데이터, 외부 언어학적 데이터베이스, 유형론적 주석이 불필요합니다.

실험 결과

  • 순위 품질: 제로샷 XLT 소스 언어 순위에서 기존 방법 대비 NDCG@3 평균 4.6% 향상 달성
  • 태스크 일반화: 다양한 NLP 태스크에서 일관된 우수성을 보여, 서브네트워크 유사도가 전이 호환성의 강건한 예측 지표임을 확인
  • 모델 내부 통찰: 광범위한 분석을 통해 다국어 모델에서 더 많은 서브네트워크 구조를 공유하는 언어 간 지식 전이가 더 효과적임을 입증

왜 중요한가?

X-SNS는 교차 언어 전이에서 소스 언어 선택을 위한 실용적이고 모델 기반의 메커니즘을 제공합니다. 외부 유형론적 지식에 의존하는 방법과 달리, X-SNS는 모델이 내부적으로 언어를 어떻게 표현하는지 직접 분석하므로 원시 텍스트 데이터가 있는 모든 언어에 적용 가능합니다. 이는 특히 모든 설계 결정이 중요한 저자원 언어에서, 올바른 소스 언어 선택이 성공적인 전이와 실패한 전이의 차이를 만들 수 있기 때문에 다국어 시스템의 효율적 배포에 큰 가치가 있습니다.
