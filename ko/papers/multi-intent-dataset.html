생략과 상호참조를 보강한 다중 의도 데이터셋

Korea Computer Congress 2024 (KCC 2024)
Sungmin So*, Jiwoo Min*, Yejin Yoon, Jungyeon Lee, Taeuk Kim

한줄 요약

생략(ellipsis)과 상호참조(coreference)를 반영한 보다 자연스러운 다중 의도 발화를 포함하는 새로운 데이터셋으로, 단순 문장 연결을 넘어선 현실적인 다중 의도 탐지 벤치마크를 제공합니다.

배경

다중 의도 탐지는 단일 발화에서 여러 사용자 의도를 식별하는 과제로, 실제 대화 시스템에서 흔히 발생하는 상황입니다(예: "서울행 비행기 예약하고 근처 호텔도 찾아줘"). 기존 다중 의도 데이터셋은 단일 의도 발화를 기계적으로 연결하여 구성되어, 실제 사람의 발화에서 나타나는 자연스러운 언어 현상이 결여된 인위적 예제를 만들어냅니다. 실제로 사람들이 하나의 발화에서 여러 의도를 표현할 때는 생략(반복 요소의 삭제)과 상호참조(이전에 언급한 개체를 대명사로 지칭)를 사용하여 발화를 더 간결하고 자연스럽게 만들지만, 이는 파싱을 더 어렵게 합니다.

방법

  • 암묵적 연결: 두 개의 단일 의도 발화를 단순히 연결하는 대신, 생략과 상호참조 해결을 적용하여 반복 정보가 삭제되거나 대명사화된 보다 자연스러운 다중 의도 발화를 생성합니다.
  • 어노테이션 파이프라인: 체계적인 어노테이션 파이프라인을 통해 결과 발화가 언어적으로 자연스러우면서도 다중 의도 및 슬롯 값이 정확하게 라벨링되도록 보장합니다.
  • 다양한 의도 조합: 다양한 도메인에 걸친 폭넓은 의도 쌍 조합을 포함하여 종합적인 평가를 가능하게 합니다.
  • 기준선 평가: 최신 다중 의도 탐지 모델을 표준 연결 기반 데이터셋과 새로운 암묵적 연결 버전 모두에서 평가하여 현실적 언어 현상의 영향을 측정합니다.

결과

기존 다중 의도 탐지 모델은 표준 연결 버전 대비 암묵적 연결 데이터셋에서 상당한 성능 하락을 보이며, 현재 시스템이 다중 의도 맥락에서의 생략과 상호참조에 어려움을 겪고 있음을 드러냅니다. 이러한 성능 격차는 인위적으로 구성된 평가 데이터에 의존하는 현재 다중 의도 탐지 연구의 중대한 사각지대를 부각시킵니다.

의의

본 데이터셋은 다중 의도 탐지 시스템 평가를 위한 보다 현실적인 벤치마크를 제공하며, 실제 사용자 발화의 언어적 복잡성을 처리하는 방향으로 연구를 이끕니다. 자연스러운 다중 의도 표현에 대한 현재 모델의 한계를 노출함으로써, 생략과 상호참조를 처리할 수 있는 보다 견고한 의도 탐지 시스템 개발을 촉진합니다 -- 이는 실용적 대화 시스템에 필수적인 능력입니다.
